IoSync — Datenpunkt-Broker

Wähle ioBroker-Datenpunkte aus, vergib einen Anzeigenamen und stelle ein Abrufintervall ein. Die Daten werden über eine HTTPS-API für Smartwatches und andere Apps bereitgestellt.

API-Server
SSL / HTTP-Modus

Aktivieren wenn Apache/nginx die HTTPS-Verbindung übernimmt (z.B. iosync.thestealth.de → localhost:7443). Die SSL-Felder unten werden dann ignoriert.

Leer lassen für automatisches selbstsigniertes Zertifikat (wird einmalig generiert und gespeichert).

API-Endpunkte:   GET /api/datapoints  |  GET /api/datapoints/{alias}  |  GET /api/health
Authentifizierung: HTTP Basic Auth mit Benutzername & Passwort
Updates via GitHub
Adapter-Updates werden über GitHub bereitgestellt. Um auf die neueste Version zu aktualisieren, führe folgenden Befehl auf dem ioBroker-Server aus:
iobroker url thestealth131205/ioBroker.iosync

GitHub-Repository: github.com/thestealth131205/ioBroker.iosync
